Designed and tested on Windows11 21h2. Based on original [Haru theme](https://github.com/LSTM-Kirigaya/typora-haru-theme).

<img width="1955" height="1149" alt="0" src="https://github.com/user-attachments/assets/6bf7e22d-2749-4d5d-ba8c-a0fdfbbf0c74" />

Haru uses [Maple Mono font](https://github.com/subframe7536/maple-font) to render code blocks and [iMWritingQuat nerd font](https://github.com/ryanoasis/nerd-fonts/releases) to render text. All fonts are bundled with `css` themes.

## Features

### Round corner design

Almost every element or container in haru is designed as round corner, including `img`, `iframe`, word count labe and even some input widget embedded in Typora.

### Readable table

Table in haru looks dark enough that the bright colors of the cells and borders don't distract you.

<img width="1596" height="972" alt="2" src="https://github.com/user-attachments/assets/8b84d26f-2e11-4328-9d69-a5a69cedb183" />


### Title + Blockquote

Combine title and blockquote to make extra information:

```markdown
> # In Haru, you can use `>` + `#` to create Tips block

> ## `>` + `##` = warning block

> ### `>` + `###` = error block

> #### `>` + `####` = remind block
```

<img width="1158" height="554" alt="4" src="https://github.com/user-attachments/assets/0d825772-70b5-4262-8d42-45fbfdda1bf2" />

Also supports GitHub alerts and notes:

```markdown
>[!caution]
> A caution

>[!note]
> This is a note

>[!warning]
> And this is a warning
```

## Install

1. Download `haru.zip` from Typora Gallery or my github [release](https://github.com/JoyHak/typora-haru-theme/releases/latest)
2. Typora Menu `->` Themes `->` Open Theme Folder 
3. Unzip `haru.zip` into the folder
4. Restart Typora
5. Select the Haru theme from the **Themes menu** in Typora
